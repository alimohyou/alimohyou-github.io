Nessus

A remote security scanning programme called Nessus looks for security flaws on a computer and alerts you if it finds any that could allow hostile hackers to access any computer you have connected to the internet. In order to accomplish this, it performs more than 1200 tests on a particular computer to determine whether any of these threats have the potential to be exploited to compromise it or cause it other harm.

OpenVAS

Similar to Nessus, OpenVAS is a system vulnerability scanner that looks for high level online threats (such as incorrect file access and cross-site script vulnerabilities) as well as known exploits on accessible services and visible ports. It is used by the TCG Tech Team to routinely scan the websites and gateway computers that we have hosted in our VMware Workstation lab.
